# Django Business Card Website

Веб-сайт визитная карточка, разработанный на фреймворке Django.

## Стек технологий
* Backend: Python 3.14, Django 6.0.5

## Реализованный функционал
* Чистая архитектура Django.
* Настроенная система маршрутизации (URL-routing).
